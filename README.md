#### 简介：
基于 Vue3 的后台管理系统，有真实后端接口，具有基本的增删查改功能，实现了按钮级，页面级，菜单级的权限控制

#### 技术栈：
Vue3.2 + TypeScript + Pinia + Tailwind CSS + Vite + Element Plus

#### 安装教程：
###### 克隆项目
`git clone https://github.com/jxz200/epidemicData-H5.git`

###### 安装依赖
`npm install`

###### 启动项目
`npm run dev`
